# AI IntelliDocs — Agentic Document Q&A System

=> An intelligent AI system that answers user queries from documents using **Retrieval-Augmented Generation (RAG)**, enhanced with **Agentic AI**, **multi-step reasoning**, and **hybrid memory**.

---

## Features

- **RAG Pipeline**  
  Retrieve relevant document chunks efficiently using **FAISS**

- **Agentic AI**  
  Automatically selects the best tool (**RAG, Web, Calculator, LLM**) based on the query

- **Multi-step Reasoning**  
  Handles complex queries like **comparison, analysis, and conversion**

- **Hybrid Memory**  
  Combines **recent conversation context** with **summarized long-term memory**

- **Tool Registry System**  
  Modular and scalable architecture for tool execution

- **Fast LLM Integration**  
  Powered by **Groq (LLaMA models)** for high-speed inference

---
